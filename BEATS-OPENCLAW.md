### **1. `computer_screenshot`** - Full Screen Capture
- Captures entire desktop (not just browser)
- Works on Windows, macOS, Linux
- Returns screenshot for vision model analysis

### **2. `mouse_click`** - Click Anywhere
- Left click, right click, double-click
- Absolute screen coordinates
- Works across all applications

### **3. `mouse_move`** - Cursor Control
- Move mouse to any position
- Hover over elements
- Precise positioning

### **4. `keyboard_type`** - Type Text Anywhere
- Types into focused application
- Special keys: Enter, Tab, Esc
- Works in any input field

### **5. `keyboard_press`** - Keyboard Shortcuts
- Ctrl+C, Ctrl+V, Alt+Tab
- System shortcuts (Win+D, Cmd+Space)
- F-keys and modifiers

### **6. `computer_use`** - High-Level API
- Combined screenshot + click + type
- Natural language: "Click the Firefox icon"
- Main Computer Use interface
